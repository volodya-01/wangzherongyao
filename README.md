###安装

项目地址：（`git clone`）

```shell
git clone https://github.com/hzlshen/vue-project.git
```

通过`npm`安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/))

```
npm install
```

启动服务(http://localhost:8080)

```
npm run dev
```

发布代码
```
npm run build
```
